Formas de ejecutar un código en JavaScript

Existen dos formas de ejecutar un código en JS, que son las que siguen

Ejecución directa:

Se produce cuando el código se ejecuta en el momento de cargar la página, sin necidad de ninguna interacción con el usuario.
Ejemplos: introducir texto mediante innerHTML, calcular la fecha, etc.

Ejecución en respuesta a un evento

Se produce cuando el código se ejecuta después de que ocurra determinada cosa, que puede o no requerir una interacción con el usuario.
Ejemplos: OnClick(con interacción con el usuario), OnLoad(sin interacción con el usuario)
